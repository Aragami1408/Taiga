# Utility Commands
## commands
### Description {docsify-ignore}
Displays all the commands available to the user
### Syntax {docsify-ignore}

> commands

### Example {docsify-ignore}

> commands

## help
### Description {docsify-ignore}
Displays a help card for the user to learn more about a specific command
### Syntax {docsify-ignore}

> help { Command Name }

### Example {docsify-ignore}

> help ping
